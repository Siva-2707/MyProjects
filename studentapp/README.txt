Prerequisites:
==============
1) Please make sure you have Java installed in your machine.

Procedure to run the application:
================================
1) Go inside target directory.
2) Run command -> 
					java -jar studentapp-0.0.1-SNAPSHOT.jar
3) Check in which port the programm is running (By default it will run at 8080)
4) Run the application in the browser with URL = localhost:8080/students
